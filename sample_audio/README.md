# Sample audio for testing

This folder holds WAV files generated by:

```bash
python scripts/generate_sample_audio.py
```

After running the script you get:

| File | Description |
|------|--------------|
| `sample_silence.wav` | 2 seconds of silence – use to test the upload and pipeline. |
| `sample_tone.wav`     | 2 seconds of a 440 Hz tone – tests non-speech input. |
| `sample_beep.wav`    | 0.5 second beep – minimal audio test. |

**To test with real speech (math question):**

1. **Record yourself:** Use your phone’s voice recorder or your PC’s mic. Say something like: *“What is the limit of x squared as x approaches 2?”* or *“Solve x squared minus 5 x plus 6 equals zero.”*
2. **Export as WAV or MP3:** Save the file (WAV works without ffmpeg; MP3 needs ffmpeg on your machine).
3. In the app: **Audio** mode → **Upload audio** → select your file → edit the transcript if needed → **Confirm and Solve**.

This directory is ignored by git by default so generated samples are not committed. Add the files to git if you want to ship them with the repo.
