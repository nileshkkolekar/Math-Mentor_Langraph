# Calculus: Derivatives

## Rules
- \( \frac{d}{dx} x^n = n x^{n-1} \)
- \( \frac{d}{dx} \sin x = \cos x \), \( \frac{d}{dx} \cos x = -\sin x \)
- \( \frac{d}{dx} e^x = e^x \), \( \frac{d}{dx} \ln x = \frac{1}{x} \)
- Product: \( (fg)' = f'g + fg' \)
- Quotient: \( (f/g)' = \frac{f'g - fg'}{g^2} \)
- Chain: \( \frac{d}{dx} f(g(x)) = f'(g(x)) g'(x) \)

## Applications
- Tangent line at \( x = a \): \( y - f(a) = f'(a)(x - a) \)
- Critical points: \( f'(x) = 0 \) or undefined. Second derivative test for max/min.
