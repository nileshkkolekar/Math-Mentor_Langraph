# Solution templates

## Quadratic equation
1. Write in form \( ax^2 + bx + c = 0 \).
2. Compute discriminant \( D = b^2 - 4ac \).
3. If \( D \geq 0 \), apply formula \( x = \frac{-b \pm \sqrt{D}}{2a} \).
4. If asked for real roots only, state "no real roots" when \( D < 0 \).

## Probability (at least one)
1. Identify complement: P(at least one) = 1 - P(none).
2. Compute P(none) using product rule if independent.
3. Subtract from 1.

## Limit (0/0 form)
1. Try factoring and canceling.
2. Or rationalize (e.g. multiply by conjugate).
3. Or use standard limit (e.g. sin x / x).
4. L'Hôpital if differentiable: limit of f/g = limit of f'/g'.
