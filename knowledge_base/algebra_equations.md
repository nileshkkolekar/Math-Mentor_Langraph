# Algebra: Equations and inequalities

## Linear equations
- \( ax + b = 0 \Rightarrow x = -b/a \) for \( a \neq 0 \)
- System of two linear equations: use substitution or elimination.

## Inequalities
- \( |x| < a \Leftrightarrow -a < x < a \)
- \( |x| > a \Leftrightarrow x < -a \) or \( x > a \)
- When multiplying both sides by a negative number, reverse the inequality.

## Domain constraints
- Denominator must not be zero.
- Expression under \( \sqrt{\cdot} \) must be \( \geq 0 \) for real solutions.
- \( \log_a x \) defined only for \( x > 0 \) and \( a > 0, a \neq 1 \).
