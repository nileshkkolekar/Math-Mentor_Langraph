# Linear algebra: Basics

## Vectors
- Dot product: \( \mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2 + \cdots \)
- \( \mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\theta \)
- Orthogonal when dot product = 0.

## Matrices
- Matrix multiplication: row by column.
- Determinant 2x2: \( ad - bc \). Inverse exists iff determinant \( \neq 0 \).
- Solve \( A\mathbf{x} = \mathbf{b} \): if \( A \) invertible, \( \mathbf{x} = A^{-1}\mathbf{b} \).

## Linear systems
- Consistent vs inconsistent (no solution). Infinitely many solutions when rank < number of unknowns.
