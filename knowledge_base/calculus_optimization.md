# Calculus: Simple optimization

## Steps
1. Write quantity to maximize/minimize as function of one variable (use constraints to eliminate others).
2. Find domain (physical or problem constraints).
3. Find critical points: set derivative to zero (and check endpoints if closed interval).
4. Evaluate function at critical points and endpoints; pick max or min.

## Common mistakes
- Forgetting to check endpoints when domain is a closed interval.
- Not verifying that the critical point is in the domain.
